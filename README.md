# struts2-freeroute-plugin #

自由的映射路由，像 Spring MVC 那样

## 引用 ##

```xml
<dependency>
    <groupId>org.apache.struts2.freeroute</groupId>
    <artifactId>struts2-freeroute-plugin</artifactId>
    <version>${version}</version>
</dependency>
```

free mapping route just like Spring MVC
